## рҹҡҖ Quick Setup

### в—Ҳ Step 1 вҖ” Get your Session ID

```
в”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғ
  рҹҢҗ  https://toosiitechdevelopertools.zone.id/session
в”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғв”Ғ
```

Pair your WhatsApp number and copy the `SESSION_ID` shown.

### в—Ҳ Step 2 вҖ” Clone & configure

```bash
git clone https://github.com/TOOSII102/toosii-xd-ultra.git
cd toosii-xd-ultra/bot
cp .env.example .env
```

Open `.env` and fill in:

```env
SESSION_ID=your_session_id_here
OWNER_NUMBER=254712345678
```

### в—Ҳ Step 3 вҖ” Install & run

```bash
npm install
node index.js
```

| Key | Value |
|:---|:---|
| `SESSION_ID` | Your session from the generator |
| `OWNER_NUMBER` | Your number (no `+` or spaces) |

| `BOT_NAME` | `TOOSII-XD` |
